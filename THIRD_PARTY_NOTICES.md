# Third-Party Notices

This file tracks notices for runtime, build, and model-development dependencies that require attribution beyond package metadata.

Runtime and development dependencies are managed through `package.json`, `pnpm-lock.yaml`, and `pyproject.toml`. Major browser runtime dependencies include React, Vite, Workbox, Playwright, and ONNX Runtime Web; their exact versions and transitive notices are resolved from the lockfiles.

## Model and data notices

- `vietasr-iter3-int8` is a metadata-only catalog entry that references external files from `zzasdf/viet_iter3_pseudo_label` on Hugging Face, pinned by revision and SHA-256 in the manifest. See `MODEL_LICENSES.md` and `docs/instructions/model-card-vietasr-iter3-int8.instructions.md` before changing or publishing any model artifact.
- `model-packs/example-manifest/files/*.onnx` are tiny deterministic CI mock graphs generated by this repository and distributed with the Apache-2.0 code license; they are not trained ASR weights.
